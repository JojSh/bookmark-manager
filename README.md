# bookmark-manager

```
As an Internet user
So that I can quickly access useful websites
I want to see a list of links to useful websites
```

```
As an Internet user
So that I can easily find the links I want
I want to sort my list of links in descending chronological order
```

```
As an Internet user
So that I can update my list of useful websites
I want to be able to add new links with their titles to the database
```

```
As an Internet user
So that I can organise my links
I want to assign category tags to my links
```

```
As an Internet user
So that I can be selective in browsing my database
I want to filter my list of links by a tag
```
